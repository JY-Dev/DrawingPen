# DrawingPen
## 제작기간 : 2018.11.13 ~ 2018.11.15
- Kotlin 이용
- 간단한 그림판 앱으로 쉽게 그림을 그리고 저장할 수 있습니다.
## 기능
- 여러가지 색 선택가능
- 이미지로 저장가능
- 펜 두께 선택가능
- 지우개 두께 선택가능

## 구현
- Canvas를 이용해 MotionEvent로 Motion UP Motion Down Event를 감지해 선을 그림
